Palindrome Checker

A palindrome is a word or phrase that reads the same when flipped backwards. A few common examples include the words "eye", "civic", and "race car". More complex palindromes may include longer phrases such as "Are we not pure? “No, sir!” Panama’s moody Noriega brags. “It is garbage!” Irony dooms a man—a prisoner up to new era."-- believe it or not, that is a palindrome!

For this section, we will be building a simple, yet quick Javascript function which will check whether a given string is a palindrome. These guides assume you already have an environment set up. If not, head over to JS Fiddle and pop in the code inside the "Javascript + No-Library" section to see it in live action.

The Code

FirstLets declare an arrow function which takes a string as an argument. Now inside the function, we take the string, and use the toLowerCase() method to convert the string to lower case. In the same line, we also include the replace() method. This method will take two parameters: a searchvalue and a newvalue. For the searchvalue, we provide the following regular expression: /[W_]/g and for the newvalue we will use an empty string. In essence, the regular expression will match for unwanted characters in the string and replace them with an empty field. For more on regular expressions, click here.
